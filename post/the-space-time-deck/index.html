Introducing the Space·Time Deck

Written by David Teller with ♥ on September 24, 2018 in rpg ronin-time-bureaucrats improv

The Space·Time Deck is a card deck designed to be used as an improv tool, both for storytelling and role-playing games.

We will be posting more about this deck in future posts.

Note that the oddly large margins are there by design. We’ll try and offer this deck for on-demand printing, and such margins are required by printers.
